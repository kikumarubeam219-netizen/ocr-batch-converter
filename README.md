# OCR一括変換アプリ

複数の画像ファイルを一括でOCR処理し、検索可能なPDFに変換するWebアプリケーションです。

## 特徴

- 🖼️ **一括画像処理** — 最大500枚の画像を同時に処理
- 🔍 **検索可能PDF** — OCRテキストを透明レイヤーとして埋め込み、PDF内のテキスト検索が可能
- 🔒 **完全ローカル処理** — すべての処理がブラウザ内で完結。画像データはサーバーに送信されません
- 🌏 **日本語対応** — Noto Sans JPフォントで日本語テキストをPDFに正確に埋め込み
- ⚡ **並列処理** — 複数のOCRワーカーによる高速バッチ処理
- 📐 **座標ベース配置** — OCRテキストを画像上の元の位置に正確に配置

## 使い方

1. 画像ファイルをドラッグ＆ドロップ（またはクリックして選択）
2. 「PDFを生成」をクリックして画像からPDFを作成
3. 「OCR処理を開始」をクリックしてテキスト認識を実行
4. 「PDFをダウンロード」で検索可能なPDFを取得

## 対応形式

PNG, JPG, JPEG, WEBP, BMP, TIFF

## 技術スタック

- **Vite** — ビルドツール
- **Tesseract.js** — ブラウザベースのOCRエンジン
- **pdf-lib** — PDF生成ライブラリ
- **@pdf-lib/fontkit** — カスタムフォント埋め込み
- **Noto Sans JP** — 日本語フォント
